# IoT_Fire-Safety (爐此安全)
## 概念
使用Flag's Block拼圖式的編譯程式來撰寫，系統可分成兩階段警報，第一階段是使用者在煮東西時，紅外線感測器會先判斷爐前是否有人，如果太久沒感測到，LED燈會開始閃爍。第二階段是溫度感測器若是感測到過高的溫度，蜂鳴器會發出警示聲，LCD也會顯示警告訊息。
