# IoT_Fire-Safety (爐此安全)
## 概念
使用Flag's Block拼圖式的編譯程式來撰寫，當使用者在燉煮東西時，系統透過紅外線感測器去判斷爐前是否有人，溫度感測器判斷溫度是否過高，並利用燈光閃爍、警示聲來吸引使用者的注意。
## 電路圖
### 紅外線感測器:偵測爐前是否有人
### 溫溼度感測器:感測爐火溫度
### LED燈:照明設備
### 蜂鳴器:警報器
### LCD螢幕:顯示溫度和警示訊息
